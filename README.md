## Environment
- python 3.8.5
- Flask 1.1.2
- SqlAlchemy 1.3.19

## Create VENV
~~~
$ python3 -m venv venv

# mac
$ source venv/bin/activate

# windows
$ source venv/Script/activate
~~~

## Install dependency packages
~~~
$ pip3 install src/requirements.txt
~~~

## Run
~~~
$ python3 src/server.py
~~~
